# StudyLink — Richesse visuelle + fonctionnalité

Cette version conserve les vraies données API et réintroduit des contenus de découverte riches lorsque la base contient encore peu d'éléments.

Principes :
- les contenus publiés dans Supabase restent prioritaires ;
- les cartes de découverte complètent les pages clairsemées ;
- chaque carte ouvre une route ou déclenche une vraie action ;
- les tutoriels de découverte utilisent de vraies vidéos YouTube intégrées ;
- les documents de découverte sont réellement ouvrables ;
- les données utilisateur privées (messages, réservations, projet) restent propres à chaque compte.
