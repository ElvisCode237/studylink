# Mon Espace d'étude

1. Exécuter `backend/migrations/015_complete_study_space.sql` dans Supabase SQL Editor.
2. Redéployer le backend Render.
3. Redéployer le frontend Vercel.
4. Ouvrir `/study-space`.

Le module contient six vues privées : Aujourd'hui, Focus, Planning, Objectifs, Notes et Statistiques.
Les premières données pédagogiques sont créées automatiquement pour chaque utilisateur lors de sa première ouverture.
