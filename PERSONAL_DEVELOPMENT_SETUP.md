# Module Développement personnel complet

## Installation des données

Option la plus simple :

1. Redéployer le backend Render.
2. Se connecter avec un compte administrateur.
3. Ouvrir `Administration > Développement personnel`.
4. Cliquer sur `Installer le module complet`.

Alternative SQL : exécuter `backend/migrations/012_complete_personal_development.sql` dans Supabase SQL Editor.

## Contenu installé

- 8 catégories
- 8 programmes guidés (14 à 30 jours)
- journées et tâches quotidiennes
- 8 guides originaux StudyLink
- 6 chapitres par guide

## Fonctions utilisateur

- filtrage par catégorie
- démarrage d'un programme
- progression personnelle
- validation des tâches quotidiennes
- tableau de bord personnel
- création et suivi d'habitudes
- lecture par chapitres
- mode nuit et taille du texte
- signets et notes privées
- reprise de lecture
